# IT Asset Checkout & Return System

A simple **Java console-based application** for managing IT assets.  
It allows adding assets, checking out assets to users, returning assets, and validating availability.

## Features
- Add IT assets
- Checkout assets
- Return assets
- Validation using custom exceptions
- JDBC-based database operations

## Technologies Used
- Java
- JDBC
- Oracle Database
- Eclipse IDE

## Project Structure
- `bean` – Model classes
- `dao` – Database operations
- `service` – Business logic
- `util` – Utility & validation classes
- `app` – Main class
